# Analytics

**Caminho:** `src/app/(admin)/admin/analytics/page.tsx`  
**Acesso:** `/admin/analytics`

## 📝 Descrição
Relatórios detalhados sobre a performance do recrutamento e conversão de candidatos.

## 🚀 Principais Funcionalidades
- **Funil de Conversão**: Taxa de passagem entre Inscritos → Triados → Indicados → Contratados.
- **Top Empresas**: Ranking de clientes que mais contratam.
- **Tempo Médio de Fechamento**: Métrica de velocidade de preenchimento de vagas.
