# Financeiro (Comissões)

**Caminho:** `src/app/(admin)/admin/finance/page.tsx`  
**Acesso:** `/admin/finance`

## 📝 Descrição
Módulo para controle de faturamento de notas fiscais e cálculo automático de comissões para os consultores.

## 🚀 Principais Funcionalidades
- **Gestão de Comissões**: Tabela com valores a receber por consultor.
- **Status de Pagamento**: Controle de faturas pendentes, pagas ou atrasadas.
- **Regras de Negócio**: Aplicação automática de porcentagens sobre o salário do contratado.
