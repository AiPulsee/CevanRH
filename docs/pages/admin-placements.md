# Contratações (Placements)

**Caminho:** `src/app/(admin)/admin/placements/page.tsx`  
**Acesso:** `/admin/placements`

## 📝 Descrição
Histórico de todas as vagas que foram fechadas com sucesso, servindo como base para o cálculo de faturamento e comissões.

## 🚀 Principais Funcionalidades
- **Tabela de Colocações**: Lista de candidatos contratados, empresa, salário e data de início.
- **Cálculo de Receita**: Visualização da receita gerada por cada contratação baseada na porcentagem acordada.
- **Filtros Temporais**: Pesquisa por período de contratação.
