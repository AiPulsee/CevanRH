# Curadoria Ativa (Managed Jobs)

**Caminho:** `src/app/(admin)/admin/managed/page.tsx`  
**Acesso:** `/admin/managed`

## 📝 Descrição
Página dedicada à gestão das vagas exclusivas (modelo Managed), onde a Cevan atua ativamente na triagem e indicação de candidatos.

## 🚀 Principais Funcionalidades
- **Lista de Vagas VIP**: Visualização de todas as vagas sob curadoria.
- **Modal de Triagem**: Acesso direto ao fluxo de análise de currículos com suporte de IA.
- **Gestão de Candidatos**: Possibilidade de indicar para o cliente, contratar ou reprovar candidatos.
- **Métricas por Vaga**: Visualização de quantos candidatos foram indicados de um total desejado (ex: 3 de 5).
