# Página Inicial Pública

**Caminho:** `src/app/(public)/page.tsx`  
**Acesso:** `/`

## 📝 Descrição
A porta de entrada para candidatos e empresas. Focada em conversão e branding da Cevan.

## 🚀 Principais Funcionalidades
- **Hero Section**: Chamada de impacto para busca de vagas.
- **Categorias**: Atalhos para áreas como Tecnologia, Vendas, Administração, etc.
- **Vagas em Destaque**: Carrossel com as melhores oportunidades atuais.
- **CTAs Corporativos**: Seções dedicadas para empresas que desejam contratar a Cevan.
