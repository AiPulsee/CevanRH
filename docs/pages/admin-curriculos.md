# Banco de Currículos

**Caminho:** `src/app/(admin)/admin/resumes/page.tsx`  
**Acesso:** `/admin/resumes`

## 📝 Descrição
Uma base de dados global contendo todos os currículos já recebidos pelo sistema, independentemente da vaga original.

## 🚀 Principais Funcionalidades
- **Busca Global**: Pesquisa por nome, e-mail ou cargo no currículo.
- **Filtros por Tipo**: Alternância rápida entre candidatos de vagas "Managed" ou "Self-Service".
- **Exportação**: Botão para exportar a lista de currículos filtrada para Excel.
- **Visualização Rápida**: Download direto do PDF ou abertura do link original.
